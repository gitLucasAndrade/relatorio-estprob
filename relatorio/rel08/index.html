1 Introdução

Na inferência estatística, o Teste de Hipóteses é uma ferramenta metodológica fundamental para a tomada de decisões e validação de suposições sobre parâmetros populacionais com base em dados amostrais. Diferente da estimação por intervalos, o teste de hipóteses avalia formalmente se uma afirmação específica sobre a população (como a superioridade de um método) possui sustentação estatística ou se a diferença observada é fruto do mero acaso.

No contexto da engenharia e otimização de processos, a comparação estatística entre diferentes configurações experimentais permite identificar os fatores que maximizam o desempenho de um sistema. Neste relatório, aplicamos esses conceitos para analisar os dados de um experimento de arremesso de bolinhas, avaliando o impacto da mudança de níveis de uma variável de controle específica na distância alcançada.

2 Objetivos

Este relatório tem como objetivos principais:

  • Formular as hipóteses estatísticas (nula e alternativa) adequadas para avaliar a igualdade das distâncias médias de arremesso entre dois grupos independentes.

  • Fixar um nível de significância (\(\alpha\)) estatisticamente robusto para guiar a região de rejeição do teste.

  • Determinar de forma matemática, a partir dos dados experimentais, qual das combinações (\(O^-\), \(A^+\), \(A^-\), \(B^+\)) proporciona a maior distância média de arremesso da bolinha.

3 Descrição do experimento

Para a execução do experimento foram usados os seguintes materiais:

  • 1 Catapulta de mesa

  • 1 Bolinha de tênis de mesa

  • 1 Trena

  • 2 Grampos de mesa

O experimento foi executado após fixarmos a catapulta em uma mesa. Uma pessoa ficou responsável pelo lançamento e pela anotação das medidas. 4 Fundamentação Teórica

  • Teste de Hipóteses para Duas Populações Independentes:

A inferência estatística divide-se classicamente em dois grandes ramos: a estimação de parâmetros e os testes de hipóteses. Enquanto a estimação dedica-se a encontrar um intervalo plausível de valores para um parâmetro populacional desconhecido, o Teste de Hipóteses formula uma conjectura prévia sobre esse parâmetro e utiliza os dados amostrais para verificar se há evidências matemáticas suficientes para rejeitá-la.

No contexto deste experimento, o objetivo não é apenas estimar a distância arremessada, mas sim decidir se a alteração do nível da variável \(B^+\) (do Nível V para o Nível VII) produz uma mudança real na média populacional ou se a variação observada entre os grupos é mero reflexo da variabilidade aleatória amostral.

4.1 A lógica do teste e a Decisão Estatística

Um teste de hipóteses possui duas afirmações mutuamente exclusivas sobre a população:

  • Hipótese Nula (\(H_0\)): É a hipótese da ausência de efeito ou da igualdade. Assume-se que qualquer diferença observada na amostra é obra do acaso.

  • Hipótese Alternativa (\(H_1\)): É a alegação que o pesquisador deseja comprovar, representando a presença de um efeito real ou diferença entre as populações.

A decisão de rejeitar ou não \(H_0\) baseia-se na fixação prévia de um nível de significância (\(\alpha\)), que representa a probabilidade máxima aceitável de cometer um Erro do Tipo I (rejeitar a hipótese nula quando ela é, de fato, verdadeira). Matematicamente:

\[\alpha = P(\text{Rejeitar } H_0 \mid H_0 \text{ é verdadeira})\]

O padrão adotado na literatura de engenharia para \(\alpha\) é de \(0,05\) (\(5\%\)). Se a probabilidade calculada de os dados terem sido gerados sob a suposição de \(H_0\) (o chamado Valor-p) menor que \(\alpha\), conclui-se que o evento é improvável demais para ter ocorrido por mero acaso, rejeitando-se \(H_0\).

  • Teste t de Student para Duas Amostras Independentes:

Sejam duas amostras aleatórias independentes de tamanhos \(n_1\) e \(n_2\), extraídas de populações normais com médias \(\mu_1\) e \(\mu_2\) e variâncias desconhecidas \(\sigma_1^2\) e \(\sigma_2^2\). Deseja-se testar a igualdade dessas médias:

\[H_0: \mu_1 - \mu_2 = 0\]

\[H_1: \mu_1 - \mu_2 \neq 0\]

Para deduzir a estatística de teste adequada, avalia-se primeiramente a relação entre as dispersões populacionais.

a) Caso Homocedástico (Variâncias Populacionais Iguais: \(\sigma_1^2 = \sigma_2^2\))

Quando assume-se que a dispersão dos dois grupos é estatisticamente idêntica, combina-se a variância das duas amostras em um único estimador ponderado pelos seus respectivos graus de liberdade, denominado variância amostral combinada (\(S_p^2\)):

\[S_p^2 = \frac{(n_1 - 1)S_1^2 + (n_2 - 1)S_2^2}{n_1 + n_2 - 2}\]

A quantidade pivotal sob a validade de \(H_0\) segue uma distribuição t de Student com \(\nu = n_1 + n_2 - 2\) graus de liberdade:

\[T = \frac{(\bar{X}_1 - \bar{X}_2) - 0}{S_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t_{n_1 + n_2 - 2}\] - Simulando matematicamente essa estrutura teórica em R:

Código 
set.seed(42)

# Simulação conceitual de duas amostras normais
n1 <- 15; mean1 <- 250; s1 <- 9.8
n2 <- 15; mean2 <- 227; s2 <- 8.0

amostra_A <- rnorm(n1, mean1, s1)
amostra_B <- rnorm(n2, mean2, s2)

# Cálculo manual da estatística T teórica (Homocedástica)
sp2 <- ((n1 - 1)*var(amostra_A) + (n2 - 1)*var(amostra_B)) / (n1 + n2 - 2)
t_calc <- (mean(amostra_A) - mean(amostra_B)) / sqrt(sp2 * (1/n1 + 1/n2))

t_calc
[1] 7.991662

b) Caso Heterocedástico (Variâncias Populacionais Diferentes: \(\sigma_1^2 \neq \sigma_2^2\))

Caso um teste prévio de igualdade de variâncias (como o Teste F deduzido no relatório anterior) indique heterocedasticidade, a suposição de uma variância combinada \(S_p^2\) torna-se inválida. Utiliza-se então a correção de Welch. A estatística de teste passa a ser:

\[T = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}}\]

Onde os graus de liberdade (\(\nu\)) são ajustados correspondentemente pela aproximação de Welch-Satterthwaite:

\[\nu = \frac{\left( \frac{S_1^2}{n_1} + \frac{S_2^2}{n_2} \right)^2}{\frac{(S_1^2/n_1)^2}{n_1-1} + \frac{(S_2^2/n_2)^2}{n_2-1}}\]

Na linguagem R, a validação de ambas as abordagens (homocedástica ou de Welch) é executada de forma nativa e unificada através do comando t.test():

Código 
# Execução direta do teste via função nativa
teste_hipotese <- t.test(amostra_A, amostra_B, 
                         alternative = "two.sided", 
                         var.equal = TRUE, 
                         conf.level = 0.95)

teste_hipotese$p.value
[1] 1.055073e-08

A fundamentação apresentada revela que o Teste de Hipóteses opera como a imagem espelhada do Estimador Intervalar: se o intervalo de confiança de \(95\%\) para a diferença das médias \((\mu_1 - \mu_2)\) não contiver o valor zero, o teste de hipóteses associado rejeitará \(H_0\) ao nível de significância de \(5\%\).

6 Conclusão

Neste relatório, exploramos a aplicação do Teste de Hipóteses como ferramenta de inferência estatística para a tomada de decisão em experimentos práticos. A partir dos dados de arremesso de uma catapulta, estruturamos uma comparação rigorosa entre duas configurações populacionais independentes para determinar o impacto da variação dos níveis da variável \(B^+\).

A fundamentação teórica permitiu compreender a mecânica do Teste t de Student e a necessidade de correções matemáticas — como a aproximação de Welch — em cenários de heterocedasticidade. A implementação computacional na linguagem R foi essencial para processar os dados coletados de forma ágil, extraindo estatísticas precisas e gerando intervalos de confiança confiáveis.

As análises responderam diretamente aos objetivos propostos: sob um nível de significância de \(5\%\), as evidências estatísticas foram suficientemente fortes para rejeitar a hipótese nula de igualdade entre as médias. Conclui-se, portanto, que a alteração da variável \(B^+\) possui impacto significativo na performance do sistema, sendo a configuração do Grupo 1 (Nível V) a mais eficiente, pois garante, em média, a maior distância de arremesso para a bolinha.

Dessa forma, o experimento reforça a importância da estatística aplicada na engenharia não apenas para descrever fenômenos, mas para otimizar processos reais baseando-se em dados e limites de confiança, minimizando a incerteza nas decisões técnicas.

Com base nos fundamentos teóricos estabelecidos, esta seção aplica o teste de hipóteses aos dados coletados no experimento de arremesso da catapulta, realizado no dia 22/06/2026. O objetivo é comparar dois grupos populacionais submetidos a diferentes configurações da variável $B^+$.

Para estruturar a análise e responder às demandas do experimento, definimos os seguintes parâmetros metodológicos:

**1 - Hipóteses do Teste (Complementares):** A hipótese nula assume que não há diferença na distância média de arremesso entre os dois grupos. Como queremos avaliar se existe uma diferença significativa (e identificar qual é a maior), utilizamos um teste bicaudal, cuja hipótese alternativa complementar é:

$H_0: \mu_1 = \mu_2$ (As médias são iguais)

$H_1: \mu_1 \neq \mu_2$ (As médias são diferentes)

**2 - Nível de Significância:** O nível de significância adotado para este experimento é $\alpha = 0,05$ ($5\%$). Isso significa que aceitamos uma probabilidade máxima de $5\%$ de cometer um erro do Tipo I (rejeitar a igualdade das médias quando elas são, de fato, iguais).

- **Simulação e cálculos em R:**

Os dados brutos coletados para ambos os grupos (com $n_1 = 15$ e $n_2 = 15$) foram transcritos e analisados computacionalmente. O Grupo 1 operou com a configuração $B^+$ no Nível V, enquanto o Grupo 2 operou com $B^+$ no Nível VII (mantendo-se $O^-$, $A^+$ e $A^-$ constantes).

Abaixo, uma implementação em linguagem R, contendo a entrada dos dados, a extração das estatísticas descritivas e a execução do teste de hipóteses.

```{r}
#| label: teste-hipoteses
#| echo: true

# Dados extraídos do experimento (Distância em centímetros)
grupo1 <- c(265, 262.5, 262, 240.5, 249, 249, 254.8, 231, 
            241.5, 248.5, 248.7, 238, 258, 257.5, 254.5)

grupo2 <- c(219, 222, 237, 222, 225.5, 232, 231, 224.5, 
            230.5, 205, 231.5, 233.5, 235, 231.5, 229)

# Aplicação do Teste t de Welch
teste_catapulta <- t.test(grupo1, grupo2, 
                          alternative = "two.sided", 
                          conf.level = 0.95)

# Organização dos resultados em um Data Frame para visualização
tabela_resultados <- data.frame(
  Med_Grupo1 = mean(grupo1),
  Med_Grupo2 = mean(grupo2),
  Estatistica_t = round(teste_catapulta$statistic, 4),
  Graus_Liberdade = round(teste_catapulta$parameter, 3),
  Valor_p = formatC(teste_catapulta$p.value, format = "e", digits = 3),
  IC_Inferior = round(teste_catapulta$conf.int[1], 3),
  IC_Superior = round(teste_catapulta$conf.int[2], 3)
)

# Renderização da tabela no relatório
knitr::kable(tabela_resultados, 
             caption = "Resultados do Teste t de Welch para Comparação de Médias",
             align = "c",
             col.names = c("Média G1", "Média G2", "Estatística t", "GL (ν)", 
                           "Valor-p", "IC Inferior (95%)", "IC Superior (95%)"))
```

- **Interpretação dos resultados:**

A análise estatística consolidada na tabela acima revela que a média amostral do Grupo 1 (250,70 cm) é superior à do Grupo 2 (227,27 cm).

Ao executar o teste de hipóteses, a estatística $t$ calculada foi de $7,1364$. O p-valor resultante ($1,139 \times 10^{-7}$) é drasticamente inferior ao nível de significância estabelecido ($\alpha = 0,05$). Adicionalmente, o intervalo de confiança de $95\%$ para a diferença das médias populacionais $[16,695 \,\, ; \,\, 30,171]$ é estritamente positivo e não contém o valor zero.

Ambas as evidências matemáticas nos levam a rejeitar contundentemente a hipótese nula ($H_0$).

Portanto, em resposta à principal questão do experimento: do ponto de vista estatístico, a combinação que proporciona a maior distância média arremessada pela bolinha é a do Grupo 1, onde a variável $B^+$ está calibrada no Nível V.
